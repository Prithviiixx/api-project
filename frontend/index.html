API #1 — Invoice Management Base URL: /api/invoices/

Creates an invoice, stores it in memory, enqueues an event to AWS SQS for async processing, and runs a background task to compute subtotal / 23% VAT / total.

Request Body application/json
{ "customer_name": "Alice Smith", // required "customer_email": "alice@test.com", // required "items": [ { "description": "Laptop", // required "quantity": 1, // int >= 1, required "unit_price": 1000.00 // float >= 0, required } ], "currency": "USD", // optional, default "USD" "notes": "Net 30" // optional }
FieldTypeRequired
customer_namestringyes
customer_emailstringyes
itemsarray (min 1)yes
items[].descriptionstringyes
items[].quantityint ≥ 1yes
items[].unit_pricefloat ≥ 0yes
currencystringno — default: USD
notesstringno
Response 200 OK
{ "message": "Invoice created successfully", "invoice": { "invoice_id": "A1B2C3D4", "customer_name": "Alice Smith", "customer_email": "alice@test.com", "items": [...], "currency": "USD", "notes": "Net 30", "status": "processed", "subtotal": 1050.00, "tax": 241.50, "total": 1291.50, "created_at": "2026-03-31T10:00:00Z" } }
200 Invoice created
422 Validation error

API #4 — Tax Calculator & Analytics Base URL: /api/

Computes subtotal, tax amount, and grand total. Default tax rate is 23% (Irish VAT). Pass any rate between 0–100.

Request Body application/json
{ "items": [ { "description": "Laptop", "quantity": 1, "unit_price": 1000.00 } ], "tax_rate": 23.0 }
FieldTypeDefault
itemsarray (min 1)—
tax_ratefloat 0–10023.0
Response 200 OK
{ "subtotal": 1000.00, "tax_rate_pct": 23.0, "tax_amount": 230.00, "total": 1230.00 }
200 Tax breakdown
422 Validation error (e.g. tax_rate > 100)

Returns a full financial breakdown for an existing invoice: subtotal → discount → taxable amount → 23% VAT → total due.

Request Body application/json
{ "invoice_id": "A1B2C3D4", "discount_percentage": 10.0 }
FieldTypeDefault
invoice_idstring—
discount_percentagefloat 0–1000.0
Response 200 OK
{ "invoice_id": "A1B2C3D4", "customer_name": "Alice Smith", "subtotal": 1050.00, "discount_pct": 10.0, "discount_amount": 105.00, "taxable_amount": 945.00, "tax_23pct": 217.35, "total_due": 1162.35 }
200 Breakdown
404 Invoice not found

Returns aggregate stats across all invoices: total count, total revenue, and status breakdown. Consumed by the dashboard stats cards at the top of the page.

Response 200 OK
{ "total_invoices": 5, "total_revenue": 6250.00, "pending": 1, "processed": 4 }
200 Stats object (always succeeds)
